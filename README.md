## 🚀 Welcome to the Chrono-Terminal
Arcoistry is not just a portfolio; it's a **multi-timeline interactive experience**. Inspired by the futuristic world of Doraemon, this site allows you to travel through different eras of my creative journey.

### 🕰️ **The Archive (Past)**
Explore the blueprints and gadgets of yesterday. Featuring a **Tokyo Rainy Night** cinematic background and interactive project cards.

### 📍 **The Hub (Present)**
The center of the 4D pocket. A dynamic space showcasing current experiments in **Chemistry, Art, and Code**.

### 🛸 **The Horizon (Future)**
A glimpse into the blueprints of tomorrow. High-speed cinematic visuals and futuristic concepts racing toward the sky.

---

## 🛠️ Secret Gadgets (Features)

| Gadget | Capability |
| :--- | :--- |
| **🚁 Take-Copter Parallax** | Experience anti-gravity scrolling. Cards float and respond to your movement as you explore the timelines. |
| **📦 4D Pocket Sync** | Automatic GitHub integration. Every time a new "gadget" (repository) is pushed to GitHub, it instantly appears in the website's Archive. |
| **🚪 Anywhere Door** | Seamless, cinematic transitions between timelines using high-fidelity video overlays. |
| **🖱️ Custom 4D Cursor** | A specialized interface tool designed for navigating high-dimensional creative spaces. |

## 🧪 Tech Stack
Built with the most advanced tools from the 22nd century:
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & Vanilla CSS
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
- **Visuals**: [Three.js](https://threejs.org/) & WebGL
- **State**: [Zustand](https://github.com/pmndrs/zustand) (Global 4D State)

## 🛠️ Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
